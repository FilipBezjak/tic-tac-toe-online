# Tic-Tac-Toe Online

Simple web Tic-Tac-Toe game (2 players or vs AI), ready to publish online.

## Run locally

Open `index.html` directly, or run a local server:

```powershell
cd C:\Users\Anže\Desktop\tic-tac-toe-online
python -m http.server 8080
```

Then open: `http://localhost:8080`

## Deploy online

## Option A: Netlify (drag and drop)
1. Go to Netlify.
2. Deploy site by dragging the project folder.

## Option B: Vercel
1. Push this folder to GitHub.
2. Import repo in Vercel.
3. Framework preset: Other.
4. Deploy.

No build step required.
